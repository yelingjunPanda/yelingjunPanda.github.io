特点

SQL语言不区分大小写。

例如:

  • create database 与 CREATE DATABASE 语句效果相同
  • books 与 BOOKS视为同一个数据库

分类

  • DDL(Database Def Language) 定义
  • DML(Database Mul Language) 增删改
  • DQL(Database Quare Language) 查
  • DCL(Database Ctrl Language) 控制

数据类型

数据类型（分有符号signed 与 无符号unsigned）
TINYINT, SMALLINT, MEDIUMINT, INT或INTEGER, BIGINT
FLOAT, DOUBLE, DECIMAL（定点数）

字符串类型
CHAR（定长,255B）, VARCHAR（变长65535B）

二进制数据

  • TINYBLOB（255 Bytes）
  • BLOB（655355 B）
  • MEDIUMBLOB（167772155 B）
  • LONGBLOB（4294967295 B）

文本数据

  • TINYTEXT（255 B）
  • TEXT（65535 B）
  • MEDIUMTEXT(167772155 B)
  • LONGTEXT(4294967295 B)
    二进制数据使用较少

日期类型

  • DATE （年月日 , YYYY-MM-DD）
  • TIME （时间 , HH:MM:SS）
  • YEAR （年份 , YYYY）
  • DATETIME （日期+时间 , YYYY-MM-DD HH:MM:SS）
  • TIMESTAMP （时间戳 , YYYY-MM-DD HH:MM:SS，但上限到2038-01-19 03:17:07）

数据类型案例：设计员工表

  1. 编号（纯数字）
  2. 员工工号（字符串，不超过10位）
  3. 员工姓名（字符串，不超过10位）
  4. 性别（男 / 女，一个汉字）
  5. 年龄
  6. 身份证号（18位，含字符X）
  7. 入职时间（年月日）
